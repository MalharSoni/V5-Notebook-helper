## Repository Purpose

This repository serves as a comprehensive resource hub for creating an Excellence Award-worthy VEX V5 Robotics Competition Engineering Notebook. It contains:

1. **Research & Documentation**: RECF judging criteria, rubrics, and award requirements
2. **Best Practices**: Structure guides, visual design principles, and winning examples
3. **Team Analysis**: Current notebook assessment and improvement recommendations
4. **Notebook Pages**: Extracted images of the current notebook (110 pages)

**Designed for**: Team members, mentors, and LLM assistants helping improve the notebook.

## Excellence Award Checklist

### Notebook Requirements
- [ ] Fully Developed status (2+ points in first 4 rubric criteria)
- [ ] 3+ brainstorm options per subsystem with diagrams
- [ ] Decision matrices for major decisions
- [ ] Multiple CLEAR design iterations
- [ ] Testing with quantitative data
- [ ] Build documentation detailed enough to recreate robot
- [ ] Sequential documentation evidence (dates, timestamps)
- [ ] All sources cited
- [ ] Student-centered (no adult contribution)

### Additional Excellence Requirements
- [ ] Top 30% of Fully Developed notebooks qualitatively
- [ ] Top 40% performance at event
- [ ] Successful team interview
- [ ] Evidence of independent inquiry
- [ ] Multi-award contender status

## Important Policies

### AI/LLM Usage (RECF Policy)
> "The use of AI/LLM programs or tools to generate, organize, enhance, or alter Engineering Notebook content or programming code is contrary to the RECF Student-Centered Policy."

**This repo is for**:
- Research and understanding requirements
- Learning best practices
- Planning and organizing approach
- Reviewing structure and gaps

**NOT for**:
- Generating actual notebook content
- Writing notebook entries
- Creating diagrams or designs

### Student-Centered Policy
- All notebook content must be student-created
- Adults can view but not edit
- Authentic student voice required
